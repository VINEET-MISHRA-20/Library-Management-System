* Library Management System*

 Description:-

A simple Java-based Library Management System that allows users to manage books, issue/return them, and store data using file handling.

# Features:-

* Add new books
* View all books
* Search books
* Issue books
* Return books
* Persistent storage using files

# Technologies Used:-

* Java
* OOP Concepts
* File Handling
* ArrayList

# How to Run

1. Compile:
   javac src/*.java

2. Run:
   java src.Main

# Project Structure:-

* Book.java → Book class
* Library.java → Core logic
* FileHandler.java → File operations
* Main.java → User interface
